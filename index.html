Info or how you can use ToBeOrNotToBe application

Chamomile tab

It is pretty simple to use it. Just drag and release leafs. You can set up the maximum amount of leafs in the application settings. Also, you can turn off leaf's sounds.

Magic Ball tab

You can start or stop ball animation by taping on it. Since version 1.0.1 you can start or stop animation by shaking the device or by covering the device's proximity sensor.

Ask Random tab

Just tap the center of the screen to get the propability of the event.

Additional info

You can disable all sounds in the application via settings.
