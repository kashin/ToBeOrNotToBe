Info or how you can use ToBeOrNotToBe application

Chamomile tab

It is pretty simple to use it. Just drag and release leafs. You can set up the maximum amount of leafs in the application settings. Also, you can turn off leaf's sounds. Use 'space' key instead of Drag'n'Drop and 'R' key to reset the flower.

Magic Ball tab

You can start or stop ball animation by taping on it or with a 'space' key. Since version 1.0.1 you can start or stop animation by shaking the device or by covering the device's proximity sensor.

Ask Random tab

Just tap the center of the screen or press 'space' key to get the propability of the event.

Additional info

Since v 1.1.0 (0.1.0 for Demo) Q10 and Q5 devices are supported, so now you can use shortcuts on these devices.
You can hide bottom bar on the Flower Page by taping on a flower's center.<> You can disable all sounds in the application via settings or by pressing 'M' on any page.
